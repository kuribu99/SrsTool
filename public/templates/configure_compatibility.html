    Compatibility refers to the non-functional aspect of the system, such as:

    1. 1. Whether the system can be executed and installed in certain operating system (Windows, Linux, Mac)
    2. 2. Whether the system can compile and run in different execution environments (Java, .NET, PHP)
    3. 3. Whether the output of the system of different version are compatible with each other (save file, generated file, image)
