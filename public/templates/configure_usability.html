    Usability refers to the effort of the system in giving the user best experience. There are few things that we can consider about usability:

    1. 1. The user interface attributes, such as the consistency of user interface
    2. 2. Providing tutorial on certain actions or features that is hard to use
    3. 3. Validating input from user to prevent user entering invalid values, such as validating email addresses
    4. 4. Preventing user from doing certain action that is error prone
    5. 5. Providing accessibility options for certain user group, such as colour blind users
